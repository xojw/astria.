# Setting Ulrua up on a server:

> [!NOTE] 
> You need a subdomain or a domain for this.

1. Log in to your server.
2. Open your server terminal.
3. Run the commands below:

```bash
sudo git clone https://github.com/xojw/astria.

sh setup.sh
```
